Homework 4

Write a brief (<200 words) essay outlining the differences between AWT and Swing graphics.

AWT is called the abstract window tool. It is a Java interface to native system GUI code present in your OS. It will not work the same on every system, although it tries. AWT components are heavyweight components. When you're using the awt components the native code used for getting the view component is generated by the Operating System, thats why it the look and feel changes from OS to OS.

Swing is part of the java foundation classes. It is a more-or-less pure-Java GUI and is 100% portable and is the same across platforms. Swing components are lightweight components, because in swing components it's the responsibility of JVM to generate the view for the components.
