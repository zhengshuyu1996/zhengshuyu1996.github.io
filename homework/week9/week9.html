Homework 1

Use Model-View-Controller to finish the survey of week 3.
>> Here is my work on port 9026

I used Node.js framework to build my server on port 9026. My MVC frame is structured as follows:
1) server: Create a server in the Node.js framework listening to port 9026. Every time the server receives a request, it will send the path name to the router to handle. Then it will continue waiting for another request.
2) router: The router receives a url and check if it is legal. Then the router will call the handler function to handle a certain url/pathname. If the handler haven't been set, that is to say the url is not legal, a response of 404 will be sent back to the client.
3) request handlers: Functions to handle the requests are defined in this part. A json file is used to save user infomation collectd from the client. And the same json will be returned when the client asks to list the results.

The client runs a javascript named main.js which sends requests and handles responses using ajax.
