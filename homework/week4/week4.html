1. Give the SELECT command to answer the question Who will go to the party held in Peking University?

A: The command should be 

SELECT party.Place, guest.Guest_Num, Guest_Name, Age, Gender, E_mail 
FROM guest, party, guest_party
WHERE party.Party_Num = guest_party.Party_Num
AND guest_party.Guest_Num = guest.Guest_Num
AND party_Name = 'Peking University'

3. Tell the difference between a CGI program and a PHP program executed by a Web Server.

A: Common Gateway Interface (CGI) offers a standard protocol for web servers to interface with executable programs running on a server that generate web pages dynamically. Though usually written in a scripting language, CGI programs can be written in any programming language.
  PHP is a free server side scripting language. It can be built into web servers like Apache and can be used to generate webpages dynamically. It's designed primarily for web development but is also used as a general-purpose programming language.We can configure Apache to load the PHP interpreter as an Apache module or run the PHP interpreter as a CGI binary.
  By default anything we type in our PHP document is given verbatim to the web browser. Each request will create a new process of the PHP interpreter. A CGI program on the other hand, is more secure because the server manages and controls access to the binaries. PHP now run as our own user rather than the generic Apache user.
